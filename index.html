Distracted Driver Detection

My First attempt at a Machine Learning model, this project contains an Image Classifier that takes drivers' images inside the car as input and outputs whether the driver is distracted or not

Blood Bank Application

This project is a blood bank application with seperate portals for Receivers and Hospitals. Hospitals can add available blood samples and check the sample requests raised by receivers. Meanwhile receivers can also view the different samples possessed by all the hospitals and can raise requests for them as long as the sample's type matches their blood type.
